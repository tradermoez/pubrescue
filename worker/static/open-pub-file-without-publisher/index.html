How to Open a .pub File Without Microsoft Publisher

Updated June 2026

Short version: only Microsoft Publisher opens .pub files natively — not Word, not PowerPoint, not Google Docs, not Preview. And with Publisher retiring on October 1, 2026, "just open it in Publisher" is about to stop being an answer for anyone. Here's what actually works.

The free-software route: LibreOffice Draw

LibreOffice (free, Windows/Mac/Linux) includes import support for Publisher files in its Draw app. Install it, then File → Open your .pub file. What to expect:

  • Simple flyers and one-pagers usually open well.
  • Fonts you don't have get substituted, which can shift text and overlap elements in dense layouts.
  • It's a ~350MB office suite install to read one attachment — worth it if you'll edit documents regularly, overkill if you won't.

What does NOT work (save yourself the search)

  • Renaming .pub to .docx/.pdf — the extension isn't the format; the file just breaks.
  • Opening in Word or Google Docs — neither has ever read Publisher format.
  • "Free .pub viewer" downloads — there is no official viewer; third-party "viewer" executables are a classic malware vector. Be careful out there.
  • Generic free conversion sites — most run the same LibreOffice engine with no preview, daily limits, and your file sitting on an anonymous server. If you try one, never upload anything confidential.
