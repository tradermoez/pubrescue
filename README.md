# PubRescue

Microsoft Publisher reaches end of life on October 1, 2026 — after that, Microsoft 365 can no longer open or edit `.pub` files.

**[PubRescue](https://pubrescue.onrender.com)** converts Publisher files (Publisher 98–2021) to PDF, PNG, SVG and editable ODG in the browser, with a free page-by-page preview before any payment.

Guides:
- [Publisher end of life: what October 2026 means](https://pubrescue.onrender.com/microsoft-publisher-end-of-life/)
- [How to convert .pub to PDF without Publisher](https://pubrescue.onrender.com/convert-pub-to-pdf/)
- [How to open a .pub file without Publisher](https://pubrescue.onrender.com/open-pub-file-without-publisher/)

Stack: FastAPI + LibreOffice (headless) in Docker, PayPal Checkout.
